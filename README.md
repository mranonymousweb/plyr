

میتوانید ویدیوی مورد نظر را با دستور 

```ffmpeg -i {ویدیو ی مورد نظر}.mp4 -profile:v baseline -level 3.0 -s 640x360 -start_number 0 -hls_time 10 -hls_list_size 0 -f hls {نام فایل}.txt```

بجای m3u8 به txt تبدیل کن

برای مثال 
# فایل هایی به نام هایی {نام ویدیو}.ts  .ts را به برای مثال .jsx تبدیل میکنیم (حتما باید یک دایرکتوری برای هر ویدیو باشد)
 بعد از این که تغییر دادید 

 میروید به {نام ویدیو}.txt و داخل سورسش
 و فایل هایی با پسوند }{نام ویدیو}.ts
 .ts را به پسوندی که در بالا تغییر دادید جای گذاری میکنید
